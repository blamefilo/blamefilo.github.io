Error Example

Solutions

  • You are using FileZilla and files have been corrupted during transfer - try using an alternative FTP client such as WinSCP
  • You are transferring the folder to your server file by file — you must upload the .zip file as-is and then extract it after it has been transferred to your VPS
  • Your server version is too old, the minimum version is 4752
    • You can download updated server artifacts here
    • ...or check out the official FiveM guide
